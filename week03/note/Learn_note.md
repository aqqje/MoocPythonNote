## 整数类型

- 可正可负, 没有取值范围限制
- pow(x, y)函数: 计算 x^y
- 十进制：1010, 99, -217
- 二进制：以 0b 或 0B 开头：0b0110, -0B0110
- 八进制：以 0o 或 0O 开头：0o545, -0O123``
- 十六进制：以 0x 或 0X 开头：0x9a, -0X78

## 浮点数类型

- 带有小数点及小数的数字
- 浮点数取值范围和小数精度都存在限制, 但常规计算可忽略
- 取值范围数量级约 -10^308 至 10^308, 精度数量级 10^-16
- 浮点数间运算存在不确定尾数, 不是bug

- round(x, d):对 x 四舍五入, d 是小数截取位数
- 浮点数间运算及比较用 round() 函数辅助
- 不确定尾数一般发生在 10^-16 左右, round() 十分有效

- 浮点数可以采用科学计数法表示
- 使用字母 e 或 E 作为幂的符号, 以 10 为基数, 格式如下:
```python
    <a>e<b>   表示a*10^b
    4.3e-3    0.0043
    9.6E5     960000.0
```

## 字符串切片高级用法

使用[M: N: K] 根据步对字符串切片
- <字符串>[M: N], M缺失表示至开头，N缺失表示至结尾
"0一二三四五六七八九十"[:3] 结果是 "0一二"
- <字符串>[M: N: k], 根据步长 K 对字符串切片
"0一二三四五六七八九十"[1: 8: 2] 结果是 "0三五七"
"0一二三四五六七八九十"[::-1] 结果是 "十九八七六五四三二一0"
